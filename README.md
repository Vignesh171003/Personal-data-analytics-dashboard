# Personal Analytics Dashboard

A Java-based Personal Analytics Dashboard that helps users track daily activities, monitor productivity, and analyze personal performance through an interactive web application.

## Features

* User Registration and Login Authentication
* Session Management
* Daily Activity Tracking
* Study Hours Monitoring
* Screen Time Tracking
* Expense Tracking
* Productivity Score Calculation
* Dashboard Analytics
* Date Range Filtering
* PDF Report Generation
* Dark/Light Theme Toggle

## Tech Stack

### Backend

* Java
* JDBC
* Servlets
* JSP

### Database

* MySQL

### Server

* Apache Tomcat 9

### Tools

* Eclipse IDE
* Git
* GitHub
* MySQL Workbench

## Project Architecture

The application follows an MVC-inspired architecture using:

* JSP for View Layer
* Servlets for Controller Layer
* JDBC for Database Access
* MySQL for Data Storage

## Database

Database Name:

personal_analytics

Main Tables:

* users
* daily_logs

## Setup Instructions

1. Clone the repository
2. Create a MySQL database named `personal_analytics`
3. Update database credentials in `DBConnection.java`
4. Import project into Eclipse
5. Configure Apache Tomcat
6. Run the application
